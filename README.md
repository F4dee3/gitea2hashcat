# gitea2hashcat
Script created in Bash to cracked the password (gitea) and export in Hashcat format.

## Instalation:

```bash
git clone https://github.com/F4dee3/gitea2hashcat
```

```bash
cd "gitea2hashcat"
```
```bash
chmod +x giteaCracked.sh
```
## Usage:

**./giteaCracked.sh -d "database.db" -o "output_file"**
